Chapitre 2:Mise en forme avec CSS

Presentation du CSS

Le role de CSS est de gérer l'apprence de la page web a travers la définition du style .

Définition des styles

Il y a 3 facons de définir des style CSS:

- Directement dans les balises du fichier html: inline
- Dans l'entete de la page : interne
- Dans une feuille de style : externe
