Syntaxe de définition des styles

Définition des styles dans les balises

On ajoute un attribut style a n'importe quelle balise.

Syntaxe:

< balise style ="propriété1:valeur;propriété2:valeur;...> </balise>

Définition des styles dans l'entéte de la page

Cela consiste a insérer le code CSS directement dans une balise <style>a l'intérieur de l'entete <head>

Syntaxe:

<head>

<style> sélecteur {propriété: valeur;propriété2:valeur;...}

</style>

</head>

Définition des styles dans une feuille de style

Les styles sont définis dans un fichier ayant pour extension .css

Syntaxe:

Il faut ajouter dans l'entéte

<link rel="stylesheet" href chemin vers le fichier.css/>
